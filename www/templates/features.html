Enter the QuoteWiz

Dispatcher Phoenix is a Konica Minolta software solution that has grown significantly over the years. With that growth, it has become a bit infamous among Konica Minolta's consultants as providing an uphill battle just to put a quote together. This particular piece of software has over 530 individual line items of licensing, maintenance, support and installation services. Needless to say, Excel just isn't going to cut it. So, here I'm introducing QuoteWiz, a semi-automated software license pricing and configuration wizard.

This application is currently deployed using open-souce code, and is primarily written in Python and HTML/JS, and is built on a PostgreSQL backend.

Downstream plans include the introduction of more configuration packages, authentication, quote tracking and recoverability, license expiration tracking A machine learning-enabled quote creation workflow, form data validation, and analytics tools are also in the pipeline.
